# A股短线主线选股看板 Pro Plus

改进：
- 个股支撑位从“20日低点”改为“近端支撑 / 防守支撑 / 趋势支撑”三层。
- 支撑位综合昨日低点、今日低点、MA5、MA10、3日低点、5日低点、枢轴点。
- 新增“短期看好个股推荐”模块，根据板块强度、个股动量、换手、成交额、价格、涨停风险自动评分。
- 推荐只做候选，不构成投资建议。

部署：
- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist
- Install Command: npm install


## v2 修正

- “当前最强主线”改成“样本最强方向”，避免误以为是全市场完整统计。
- “涨停数”改成“样本涨停数”。
- 样本涨停数按主板10cm、创业板/科创板20cm、ST约5cm近似识别。
- 增加统计口径说明。
- 如果前两大板块评分接近，提示多主线轮动。


## v3 完整板块数据

新增：
- `/api/boards`：读取东方财富概念/行业板块列表。
- `/api/boardStocks`：读取指定板块完整成分股。
- 网页新增“完整板块数据：概念/行业全量成分”模块。
- 可查看所选板块成分股数量、上涨占比、涨停数、强势数、成交额、成分涨幅榜和板块内短线候选。
